git init . 	            - > cria um repositório git na pasta atual
git status	            - > mostra o status de trabalho (cada arquivo)
git add nome_arquivo	- > adiciona arquivo para ser commitado
git commit -m "msg"	    - > faz commit do que estava para ser commitado 
git log                 - > mostra todos os históricos de commit

## FLUXO DE TRABALHO
1. criar o diretório (pasta) do projeto - mkdir
2. iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome_arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta pro passo 3.
